# PROMPTS.md

This file lists the prompts used when generating code with AI tools, as required by Cloudflare.

---

## Prompt 1: Project Structure
"Give me a modern Cloudflare Workers AI project layout using Agents, Durable Objects, and a clean API."

## Prompt 2: Memory System
"Write a Durable Object implementation that can save and load JSON conversation memory."

## Prompt 3": Agent Actions
"Generate actions for lyric generation, track naming, and Armonia help using Llama 3.3."

## Prompt 4: REST API
"Create API routes for chat, lyrics, and help that call agent.run(action)."

## Prompt 5: UI Demo
"Create a minimal HTML/CSS/JS chat UI that posts to `/api/chat`."

## Prompt 6: C# Integration
"Make an HttpClient-based C# integration class to call the Cloudflare Worker."
